# 科幻「群星彼端」机缘体系 设计（spec）

> 把旗舰 xian 已验证的「机缘体系」铺到第 5 个题材 科幻。**核心立意：世代飞船的本质是「能不能抵达」的旅程，不是个人练级**（三国教训：先想本质张力，别套模板）。脊梁＝**航程阶段**，区别于其它题材的单系统升级。

## 现状（scifi.ts，1604 行）
- 3 属性：`tech 科技`(init15)、`integrity 船体`(init75, deathBelow0, decay3)、`colony 文明火种`(init50, deathBelow0, 无 decay)。
- 3 开局：殖民舰长 / 首席科学家 / 临危受命的代理舰长——**均无 flag**。
- **无** ceilingUnlocks、无 tierLabel、无 flagsSet/flagsClear/endTone。
- ~80 个 localEvents（含 keyMoment、once、itemsGained 外星造物/生态种子库/星图残片/黑石碎片/舰宠鼠 等因果种子）。
- ~25 个结局，多为 maxTurns 门控的 tech/integrity/colony 组合 + 几个结构哨兵（colony<=6、integrity<=6 等）。

## 设计总览（航程为脊）

### 1. 进阶脊梁：航程阶段（挂 `科技` ceilingUnlocks，只进不退）
飞船每跨越一段航程，获得新资源/造物/星系能源，**科研天花板随之抬升**——硬科幻自洽，且科技奇点必须走完旅程才摸得到。

```
tech: { initial:15, max:100,
  ceiling: 25,                       // base 须 ≥ initial（=「捉襟见肘」带上限）；深空航行初期资源匮乏锁死高科技
  ceilingUnlocks: [
    { flag:'深空', max:45 },          // 立足深空、自给自足成形
    { flag:'越障', max:70 },          // 穿越星际险区（陨石带/辐射区）
    { flag:'抵近', max:90 },          // 抵近目标星系、锁定新家园
    { flag:'扎根', max:100 },         // 登陆殖民应许之地
  ] }
```
- tierLabel = **航程**。AI【晋阶之序】将叙述「本剧航程依次为：深空→越障→抵近→扎根」。
- **航程只进不退**（无 flagsClear）——旅程不会倒流；张力由 `船体`衰减 + `文明`危机 双死亡线承担（科幻的独特点）。

### 2. 四道升程闸门（keyMoment + flagsSet，按序串链 has(prev)）
**优先改造既有事件**（复用文案、避免 summary 撞名——梨园教训）。各闸门由**「掌握/突破」那一支**授印记（叙事＝勉强渡过 vs 真正掌握并打开新科研维度）；另一支只是渡过、不晋阶。

| 印记 | 改造既有事件 | 新增门控 `requires` | 授印记的选项 |
|---|---|---|---|
| **深空** | `水源告急`(minTurn3) | （无新增；加 keyMoment） | 「研发新型水循环装置」→ flagsSet:['深空']（建立可持续生命维持＝立足深空）|
| **越障** | `陨石带`(weight1.2) | `has(深空)`（加 keyMoment）| 「研发临时护盾再穿越」→ flagsSet:['越障'] |
| **抵近** | `候选行星`(keyMoment,minTurn18) | `has(越障)` | 「改道详查，或是新家园」→ flagsSet:['抵近'] |
| **扎根** | `第一座城`(minTurn27, requires colony>=50) | `has(抵近) & colony>=50`（加 keyMoment）| 「开放包容，广纳人心」→ flagsSet:['扎根'] |

- **同回合即时生效**：引擎 applyChoice 先应用 flags 再 clamp，故授印记当回合 tech 即可破旧上限（沿用引擎既有行为，建测试覆盖）。
- 串链保证顺序：`陨石带` 即便无 minTurn 也因 `has(深空)` 不会先于 `水源告急` 触发。
- **sim 杠杆**：若某闸门的晋阶支被策略极少选中、导致登顶率≈0，则改为两支都授印记（叙事＝两条路都渡过且各有所得）。spec 默认单支授印记，sim-balance 阶段校准。

### 3. 三开局身份印记（新增，不删既有内容）
开局加 `flag`，并**新增 3 道身份事件**（各 `requires: has(<flag>)`，weight~0.9），捕捉各开局的招牌张力——新增而非改造既有事件，避免把通用危机从其它开局剥走。

| 开局 | flag | 新增身份事件主题 |
|---|---|---|
| 殖民舰长 | `殖民舰长` | 一言九鼎的孤独——绝对权威既是底气也是众矢之的（决断 vs 纳谏）|
| 首席科学家 | `首席科学家` | 生存与伦理的私人煎熬（以技救船 vs 守住科学良知）|
| 临危受命的代理舰长 | `代理舰长` | 根基未稳、旧部不服——立威 vs 怀柔以收人心 |

### 4. 双死亡属性张力（已大体存在，sim 校准）
- `船体` decay 3：悬顶之危（attrition，须主动维修对冲）——主死亡线，保持。
- `文明火种` decay：**起始 0**，sim-balance 阶段按死亡率定（三国教训：事件普遍+属性、decay 须够大才咬得动）。若 文明断绝 几乎不可达则加 1–2。
- 立意：船体＝必须对冲的熵增；文明＝必须斡旋的人心危机。双线皆可致死，是科幻区别于单一死亡线题材的核心。

### 5. 隐藏 endTone 彩蛋（哨兵 `colony<=-1`，靠事件 endTone 强制；2 地狱 + 1 天堂）
新增 3 个结局，`condition:'colony<=-1'`（永不自然为真的哨兵守卫，仅由某事件选项/子结局的 `endTone` 强制触发）。挂在既有极端抉择的**子结局**上（沿用三国 功高震主 outcomes[].endTone 模式）：

| 基调 | 性质 | 宿主事件·支 |
|---|---|---|
| **清洗续命·血债驶向虚空** | 地狱 | `加密屏前的减员名单`「默许 AI 的冷酷算计」支 → 子结局 endTone |
| **屠灭原民·新世罪基** | 地狱 | `原住文明`「掠夺其资源，先下手为强」支 → 子结局 endTone |
| **星海共生·文明永续** | 天堂 | `外星问候`「谨慎回应，尝试交流」支 → 稀有正向子结局 endTone |
- 哨兵基调均用 `colony`（弃舟/清洗/屠民/共生 皆系于「人/文明火种」）。
- 测试：每个哨兵基调都被某事件 endTone 引用（防 tone 打错——三国同款守护）。

### 6. apex 门控（无需改结局——ceiling 自动达成）
- 航程印记锁 `科技` ceiling，故既有高科技结局**自动**隐式要求走完旅程：
  - `技术奇点·星海新生`(tech>=96) ⟺ 必有 `扎根`（ceiling100）；
  - `抵达新世界·开创纪元`(tech>=80) ⟺ 必有 `抵近`（ceiling90）；
  - `万世昌隆·星海帝国`(tech>=60) ⟺ 必有 `越障`（ceiling70）。
- 航程只进不退（无 flagsClear），故 tech 高 ⟺ 已抵该印记，**无需在结局上加 has() 兜底**（区别于三国须 has(霸业) 防失势）。保持结局现状，仅靠 ceiling 静默门控。
- 不进则结局封顶：停在 深空（ceiling45）者只能落 tech≤45 档/纯 colony 档/纯 integrity 档及兜底结局——「不经营航程＝得次等结局」，符合机缘立意。

### 7. AI 注入 & systemPrompt
- 加 `tierLabel:'航程'` → prompt.ts 既有【晋阶之序】自动用本剧术语 + 航程印记序（无须改 prompt.ts，已通用化）。
- systemPrompt 补一句：航程阶段由真机缘（重大航段抉择）推进、科技天花板随航程抬升；极端泯灭人性/僭越之举可能触发隐藏结局。**不讲数值/封顶**（引擎 clampEffects 静默强制）。

### 8. sim-balance
跑 `npx vite-node scripts/sim-balance.ts scifi 5000`（random/survive/greedy）。校准目标：
- 死亡率健康（船体解体 + 文明断绝 + endTone 致死 合计非 0 也非过高）；
- 登顶（抵近/扎根 + 高 tech 档）random 稀有但 greedy/survive 可达（非 0）；
- 据此定 `文明火种` decay 与（必要时）闸门是否双支授印记。

## 不做（YAGNI）
- 不加航程 flagsClear/倒退（旅程单向；volatility 是三国的本质，非科幻）。
- 不重写既有 ~80 事件/~25 结局的文案；只做最小机缘叠加（印记/门控/身份/哨兵/tierLabel）。
- 不动 maxTurns（30 年航程已合适）。
